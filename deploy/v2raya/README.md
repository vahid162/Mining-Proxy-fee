# v2rayA data directory

این پوشه به‌صورت volume در `docker-compose.yml` به مسیر `/etc/v2raya` داخل کانتینر mount می‌شود.

هدف:
- نگه‌داری تنظیمات/state مربوط به v2rayA روی host
- ماندگاری تنظیمات بعد از restart/recreate کانتینر

نکته عملیاتی:
- خالی بودن اولیه این پوشه طبیعی است؛ v2rayA در اولین اجرا فایل‌های لازم را می‌سازد.
- این مسیر را حذف نکنید مگر اینکه قصد reset کامل تنظیمات v2rayA را داشته باشید.
